# Source Boundary

This skill is a Codex-native coordination layer primarily inspired by `pamirtuna/gamestudio-subagents` and extended with user-requested workflow rules.

It may refer to local skills such as `ponytail`, `generate2dsprite`, and `generate2dmap` as workflow inspiration, but it should not vendor their scripts, assets, or full documentation unless the owner intentionally copies and attributes them.

Public source references:

- `https://github.com/pamirtuna/gamestudio-subagents`
- `https://github.com/DietrichGebert/ponytail`
- `https://github.com/0x0funky/agent-sprite-forge`

For public publishing:

- keep the skill name `gamestudio`
- keep attribution to `pamirtuna/gamestudio-subagents`, `DietrichGebert/ponytail`, and `0x0funky/agent-sprite-forge`
- do not claim to include or execute Claude Code sub-agent runtime behavior
- document original additions as Codex workflow guidance, not copied source
- preserve license and notice files when publishing the repository
